# mini-raytracer
A cpu-based raytracer written in rust. This project was written by me to learn about computer graphics, specifically rayracing. 

# features
* Parallelized frame computation.
* Diffuse reflection.
* Specular reflection.
* Shadows.
* Recursive object reflections.
